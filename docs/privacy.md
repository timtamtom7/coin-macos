# Coin Privacy Policy

**Last updated: 2026-03-25**

## Overview

Coin is a local Mac security audit tool. All data processed by Coin stays on your Mac.

## Data Collection

Coin performs local security checks only. It does not:
- Collect personal information
- Send data to any server
- Use analytics or tracking
- Store credentials or sensitive data

## Security Checks

Coin checks the following macOS security features:
- Gatekeeper status
- Firewall status
- FileVault encryption status
- System updates
- Security preferences

Results are displayed locally and may be stored in a local SQLite database on your Mac.

## No Third-Party Services

Coin does not use any third-party SDKs, ad networks, or data brokers.

## iCloud Sync

If iCloud sync is enabled, only non-sensitive preferences (settings, watchlist) sync via your Apple ID. Security scan results are never synced.

## Widgets

Widgets display cached security scores only. No personal data is shared with widgets.
